CTBS { Prototype }

Atualmente em desenvolvimento


  CTBS { Prototype } é um sistema de batalha para o RPG Maker VXAce inspirado no clássico Chrono Trigger. Apesar do sistema manter várias das características que tornaram o sistema deste jogo tão icônico, ele adiciona novas funcionalidades e corta outras, trazendo uma nova roupagem ao sistema.
